Financial Dashboard (Vanilla JavaScript)
A modern, responsive Financial Dashboard Web App built using HTML, CSS, and Vanilla JavaScript.
It helps users track income, expenses, and financial insights with interactive charts and filters.

🚀 Features
📊 Dashboard Overview
Total Income, Expenses, and Balance
Real-time calculations
Clean card-based UI

📈 Charts & Visualization
Balance Trend (Line Chart using SVG)
Spending Breakdown (Pie Chart)
Dynamic updates based on transactions

💳 Transaction Management
Add, Edit, Delete transactions
Categorize income & expenses
Fully interactive table

🔍 Advanced Filters
Filter by:
Type (Income/Expense)
Category
Date Range
Sorting:
Date (Asc/Desc)
Amount (Asc/Desc)

🌙 Dark Mode
Toggle between Light & Dark theme
Smooth UI transitions

👤 Role-Based Access
Viewer → Read-only access
Admin → Full access (Add/Edit/Delete)

📤 Export Data
Export transactions as:
CSV
JSON

💡 Insights
Highest spending category
Monthly expense comparison
Income vs Expense ratio

💾 Local Storage
Data persists in browser
No backend required

📂 Project Structure
📁 financial-dashboard
 └── Dashboard.html

⚙️ How to Run
Download or clone the repository
Open the file:
index.html
Run in any browser (Chrome, Edge, etc.)
